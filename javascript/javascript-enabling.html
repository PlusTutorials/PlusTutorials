Enabling JavaScript in Browsers

⬅ Previous Page
Next Page ➡

Javascript is built-in and available in all the modern browsers. Browsers give the option to enable or disable JavaScript manually. In this tutorial you will learn how to enable or disable JavaScript support in your browsers: Internet Explorer, Firefox, chrome, and Opera.

JavaScript in Internet Explorer

Follow these steps to enable or disable JavaScript in your Internet Explorer:

1. Go to Tools -> Internet Options from the menu.
2. Select Security from the dialog box.
3. Click on the Custom Level button.
4. Scroll down to Scripting option.
5. Select Enable radio button under Active scripting.
6. Click OK.

To disable JavaScript in Internet Explorer, select Disable radio button under Active scripting.

JavaScript in Firefox

Follow these steps to enable or disable JavaScript in your Firefox:

1. Open a new tab and type about: config in the address bar.
2. You will then get prompted by the dialog. Select I'll be careful, I promise!
3. You will then find the list of configure options in the browser.
4. In the search bar, type javascript.enabled.
5. You will then find the option to enable or disable javascript by right clicking on the value of that option -> select toggle.

Upon clicking toogle, if javascript.enabled is true, it converts to false. If it is disabled, it converts to enabled upon toogle.

JavaScript in Chrome

Follow these steps to enable or disable JavaScript in your Chrome:

1. Click on the menu button at the top right corner of chrome browser.
2. Select Settings.
3. Click Show advanced settings at the end of the page.
4. Under the Privacy section, click the Content settings button.
5. In the "Javascript" section, select "Do not allow any site to run JavaScript: or "Allow all sites to run JavaScript (recommended)".

JavaScript in Opera

Follow these steps to enable or disable JavaScript in Opera:

1. Follow Tools -> Preferences from the menu.
2. Select Advanced option from the dialog.
3. Select Content from the listed items.
4. Select Enable Javascript checkbox.
5. Click OK.
