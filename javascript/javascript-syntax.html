Whitespace and Line Breaks

It is important to note that Javascript ignores spaces, tabs, and newlines. Therefore, they can be used to format your code in an understandable and easy to read style.

Semicolons are Optional

In the JavaScript world, semicolons are optional to use and can be omitted if each of your statements are placed on a separate line, however it is recommended as best practice to place a semicolon after each statement:

Case Sensitivity

It is important to note that JavaScript is a case-sensitive programming language. Function names, keywords, and other variables must always be typed with a consistent capitalization of letters.
