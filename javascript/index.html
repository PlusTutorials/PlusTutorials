Javascript is the programming language of the Web. It is a high-level, interpreted programming language characterized as dynamic, weakly typed, protype-based and multi-paradigm. Javascript is easily implemented because it is integrated with HTML.

Audience

This tutorial is designed to help beginners as well as professionals who want to learn and understand the basic functionality of JavaScript to build dynamic web pages and applications.

Prerequisites

For this tutorial, you should be familiar with developing simple Web Pages using HTML. It is also recommended if the user had prior exposure to object-oriented programming concepts and a general idea on creating online applications.
