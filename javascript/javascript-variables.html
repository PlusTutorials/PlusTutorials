JavaScript Datatypes

There are three different primitive data types available in JavaScript:

1. Numbers, e.g. 1337, 27.50 etc.
2. Strings of text e.g. "This text is a string" etc.
3. Boolean e.g. true or false.

Furthermore, JavaScript supports two important data types, null and undefined. Javascript additionally support a data type known as object.

JavaScript Variables Scope

The variable scope refers to the location where it is defined. There are two scopes available for JavaScript variables:

1. Global Variables - are variables that can be defined anywhere in the JavaScript code.
2. Local Variables - are variables that are defined within a function.
