JavaScript - Placement in HTML File

⬅ Previous Page
Next Page ➡

JavaScript can be enabled anywhere in an HTML document. It is best recommended to include JavaScript in the following locations:

1. Script in <head> ... #&60;/head> section.
2. Script in <body> ... </body> section.
3. Script in <body> ... </body> and <head> ... </head> sections.
4. Script in an external file and then include in <head> ... </head> section.
