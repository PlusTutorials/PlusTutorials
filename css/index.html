CSS also known as Cascading Style Sheets was initially developed for the purpose of styling and describing the presentation of a document written in a markup language like HTML. This tutorial gives a complete understanding of CSS, starting from the basics to advanced concepts.

Audience

This tutorial is designed to help beginners as well as professionals who want to learn how to make their websites or personal blogs more attractive using CSS.

Prerequisites

You should be familiar with developing simple Web Pages using HTML, Internet browsers, and basic use of any text editor.
