CSS - Text

⬅ Previous Page
Next Page ➡

This tutorial explains how to use CSS text properties to edit and modify text in an HTML document. Use the following CSS text properties to edit text elements:

  • 1. color property to set the color of a text.
  • 2. direction property to set the text direction.
  • 3. letter-spacing property to modify the amount of space between the letters in text.
  • 4. word-spacing property to modify the amount of space between the words in text.
  • 5. text-decoration property to overline, underline, and strikethrough text.
  • 6. text-align property to align the text of a document.
  • 7. text-indent property to indent the text of a document.
  • 8. text-transform property to capitalize text, or convert it to uppercase or lowercase letters.
  • 9. white-space property to control the formatting and flow of text.
  • 10. text-shadow property to set the text shadow.
