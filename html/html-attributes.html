HTML - Attributes

⬅ Previous Page
Next Page ➡

HTML Attributes are used to give HTML tags extra bits of information. They define HTML tags characteristics and placed inside the element's opening tag. All attributes are made up of two parts - a name and a value.

The name is the property you want to set. As shown in the example below, the < element consists of an attribute whose name is align (used to define the alignment of the paragraph on the page).

The value points to the value of the property to be set and must be put within quotations. It is important to note that attribute names and values are case-insensitive.
