HTML - Colors

⬅ Previous Page
Next Page ➡

The following attributes are available to use with the body tag to set different colors at the page level:

  • 1. bgcolor - sets the color of the background page.
  • 2. text - sets the color of the body text.
  • 3. alink - sets the color of the active or selected links.
  • 4. link - sets the color of the linked text.
  • 5. vlink - sets the color of the visited links.

HTML Color Coding Methods

The following three methods are used to set colors in your webpage:

  • 1. Color names - specify color names like green, blue or red.
  • 2. Hex codes - a six-digit code specifying the amount of red, green and blue combination that makes up the color.
  • 3. Color decimal/percentage values - this value is specified using the rgb() property.

HTML Colors - Hex Codes

A hexadecimal is a six-digit code specifying the amount of red, green and blue combination that makes up the color. The first two digits(RR) represent the red value, the next two represent the green value(GG), and the last two represent the blue value(BB). The following are examples of colors using hexadecimal notation:
