ASP.NET is a server-side web application framework built by Microsoft for web development to produce dynamic web sites, web applications, and web services. It helps you build web applications easily using C#. This tutorial has been designed to cover all the fundamental elements of ASP.NET that you would require to get started.

Audience

This tutorial has been prepared for the beginners to help them understand the basic to advanced concepts related to ASP.NET.

Prerequisites

Before you start this tutorial, we assume that you have a basic understanding of .NET framework. In this tutorial we are going t build web-based applications using ASP.NET web application framework, so it would be a good idea to have an understanding of HTML, CSS, AJAX, etc.
