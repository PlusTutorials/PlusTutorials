ASP.NET is a server-side web application framework built on top of the HTTP protocol to build up dynamic web applications. ASP.NET is a part of Microsoft .Net framework, and written using the reusable elements or objects preset in .Net framework. ASP.NET applications can be written using any of the following programming languages:


  • C#
  • Visual Basic.Net
  • Jscript
  • j#

ASP.NET consists of a large number of components such as buttons, text boxes, and labels for configuring, assembling, and manipulating code to create interactive, data-driven web applications over the internet.
