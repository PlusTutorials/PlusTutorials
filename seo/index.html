SEO stands for Search Engine Optimization is the process of optimizing a website or a web page to increase its visibility (higher positions in search results) to users of a search engines such as Google, Bing, Yahoo, etc. and enable it to generate more traffic.

In this tutorial we will explain the tips and tricks of effective SEO characteristics.

Audience

This tutorial has been prepared for the beginners to help them understand the basic to advanced concepts related to effective SEO characteristics.

Prerequisites

Before you start this tutorial it is recommended to be aware of simple web technologies such as HTML, CSS, etc. Having your own website will be an added advantage as it will allow you to apply the SEO concepts in this tutorial.
