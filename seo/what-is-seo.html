SEO - What is SEO?

Search Engine Optimization is a process to opimize a web page or website for search engines. SEO is a technique for:

  • Increasing the traffic to a web page or website generated by search engine results.
  • Understanding how search engines algorithms work and marketing your website's content accordingly
  • Building a web page or website to rank (index position) well in search engines (Google, Bing, Yahoo, etc.) results.

Since most of the techniques used in SEO deal with text, it is also reffered to it as SEO copyrighting. It is important to understand how search engines algorithms work to be able to perform basic SEO.

How Search Engines Work?

To deliver search results, search engines perform several techniques and processes.

  • Crawling - Special Software known as a crawler or a spider performs a technique of fetching all the web pages linked to a website.
  • Indexing - A process of storing all the fetched web pages from crawling and storing them in a database to be later retrieved by linking the web pages to a particular set of keywords.
  • Processing - Search engines perform a process of matching the search query request with the indexed web pages in the database.
  • Calculating Relevancy - Search Engines run their algorithm on each of the indexed web pages to determine their relevancy to the search query request and rank them accordingly.
  • Retrieving Results - The last step in the process is retrieving the best determined results and displaying them to the user.

What is SEO copywriting?

SEO copywriting is a technique of writing useful and valuable text that targets a specific set of search terms and keywords. Additionally, SEO copywriting is used to optimize elements such as Description, Title, Headings, Keywords tags, and alternative text. Search engines look for valuable content pages which is the idea of SEO copywriting.

What is Search Engine Rank?

Search engines web pages ranking is measured by the position of displaying a web page. When a search engines receives a search query request, the search engine displays thousands of results from the archived web pages in their databases. If your web page is getting displayed on the first position, then it's rank is 1 and will be determined as the web page with the most ranking.
